# Data-Structures Module

This repository contains Python code to simulate Data Structures algorithms to understand how common data structures and techniques work.

## Included Topics

- **Huffman Coding** – implementation of a compression-related tree-based encoding approach
- **Hashed Dictionary** – exploration of hashing and dictionary-style key-value storage
- **Tower of Hanoi** – recursive problem-solving
- **Integer Sorting** – basic sorting logic
- **Exponentiation / other small exercises** – additional algorithm practice
